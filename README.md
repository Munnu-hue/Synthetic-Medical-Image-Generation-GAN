# Synthetic Medical Image Generation using GAN

## Project Description
This project focuses on generating synthetic medical images using Generative Adversarial Networks (DCGAN and CGAN). The generated images help increase dataset diversity and improve the performance of deep learning models used in medical image analysis.

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Models Used
- DCGAN
- CGAN
- Traditional Data Augmentation
